How to install the stack of tasks on your machine?

Prerequisites

The current software architecture has been tested entirely only on Ubuntu 10.04 LTS and 12.04 LTS. A large part of the basic algorithms have been tested on Windows and Mac, but there is no guarantee that it is still true.

There is currently no up-to-date binary distribution.

The installation is done through pulling and compiling a dozen of C++ repositories.

Dependencies

The needed software are: git, CMake, boost 1.40.0 / 1.46.0, lapack, blas, python 2.6 / 2.7
  • CMake
  • gcc, g++
  • Doxygen (and eventually LaTeX for mathematical formula rendering)
  • Boost
  • Eigen
  • Blas, lapack
ROS (fuerte, groovy or hydro). Please note that most of the packages are ROS-independent, but some packages allow to link the SOT framework to ROS (e.g. redundant_manipulator_control)

Binary installation

Not available in its current state.

Source installation

The sources can be installed using the script available at the following repository: install-sot.
